# DeepAgent Research Delegation

## Overview

This skill standardizes how the agent performs delegated research and integrates results.

## Instructions

1. Create one in-progress todo item that names the research target.
2. Delegate with `task` and provide a narrow goal and expected output format.
3. Validate returned evidence before using it.
4. Convert accepted findings into implementation steps and mark todos.
5. If evidence is weak or conflicting, request a second delegated search.
